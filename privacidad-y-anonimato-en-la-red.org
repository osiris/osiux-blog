* tunnel SSH

  #+begin_example
    ssh -L 8080:osiux.com:80  osiris@osiux.com
           ^    ^         ^    ^      ^
           |    |         |    |      |
           |    |         |    |      servidor
           |    |         |    |
           |    destino   |    usuario
           |              |
           puerto local   puerto destino
  #+end_example

* siempre =localhost=

  - aplicaciones apuntan a =localhost:puerto=
  - si el =tunnel ssh= no está levantado, no hay canal *seguro*

Gracias al post de *lukio* [fn:lukio] basado en el blog de *ubuntu
peronista* [fn:ubuntuperonista] pude hacer funcionar el sintonizador
*TV Digital Geniatech USB MyGica S870* con Sidux [fn:sidux] (Debian
sid).

Los pasos a seguir son los siguientes

   #+BEGIN_EXAMPLE
    sudo apt-get install mercurial linux-headers-$(uname -r) build essential
    cd ~
    hg clone http://linuxtv.org/hg/v4l-dvb
    cd v4l-dvb
    make
    sudo make install
   #+END_EXAMPLE

Si falla la compilación, debemos editar el =.config= y cambiar la
siguiente linea:

   #+BEGIN_EXAMPLE
     vim v4l-dv/v4l/.config
     CONFIG_DVB_FIREDTV=m y la cambiamos a CONFIG_DVB_FIREDTV=n. Luego guardamos y cerramos el archivo.
   #+END_EXAMPLE
